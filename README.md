# Will You Please

Will you please is a script runner for local development.

## Installation

```bash
go get -u github.com/gschier/wyp
```

## Usage

```bash
# Generate a wyp.yaml file to put scripts in
wyp init

# List runnable scripts
wyp run

# Run scripts
wyp run start
wyp run greet

# Shorthand for running start script
wyp start
```
